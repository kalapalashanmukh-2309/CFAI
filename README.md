# Sorting Algorithm Performance Comparator
📌 Overview

The Sorting Algorithm Performance Comparator is a full-stack web application that allows users to analyze and compare the execution time of different sorting algorithms in real time.

Users can input a custom dataset, and the system processes it using multiple algorithms such as Bubble Sort, Merge Sort, and Quick Sort, displaying their execution times instantly.

This project is designed to help students and developers visualize algorithm efficiency and understand time complexity in a practical way. 
# Features
. Custom input support (comma-separated numbers)

. Real-time performance benchmarking
 
. Compare multiple sorting algorithms

. Helps understand DSA concepts visually

. Simple and interactive UI
